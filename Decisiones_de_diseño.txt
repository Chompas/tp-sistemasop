Decisiones de implementacion:

- Instalar_TP
	El log de instalación es único. Todas las instalaciones loguearán en el mismo archivo dentro de $CONFDIR
	Recibe un único parámetro con la ruta del GRUPO.
	Ej: ./Instalar_TP.sh "path/hasta/el/grupo"

- Grabar_L
	Un parametro opcional "-i" para invocarlo solo desde el modulo de instalación
	Cuando el archivo de log es muy grande se quitan todas las lineas, excepto las ultimas 5
	Fecha y hora en formato DD-MM-AA HH:MM:SS (script aparte?)

- Mover_A
	El numero de secuencia nnn es descentralizado (no es comun para todo el sistema), 
	se lo obtiene directamente del directorio de duplicados, por eso no es necesario guardarlo

- Imprimir_A
	Se usa un array asociativo (estructura de hash) para:
		- Manejar las opciones y las funciones a las que estan asociados esas opciones.
		- Manejar los eventos candidatos para la opción "-i"
	Si se especifica la opción "t", el archivo se escribe siempre, independientemente de si se especificó "-w" como argumento.
	Para chequear que no haya una instancia en ejecución, se utiliza un archivo temporal Imprimir_A.PID que se crea al crear una instancia y se elimina al terminar la ejecución.
